# Plant-Disease-Data-Analysis-using-Python

This project focuses on analyzing a plant disease dataset using various data science techniques. The goal is to understand patterns in plant health data, identify relationships between features, and apply machine learning models for prediction.

The dataset contains information related to plant conditions and disease indicators. Using Python and popular data science libraries, the project performs end-to-end analysis including data preprocessing, statistical analysis, visualization, and predictive modeling.

🔧 Features Implemented 
Data Cleaning:
Missing values are handled using appropriate techniques such as mean replacement. Irrelevant or non-numeric data is filtered to ensure accurate analysis.
Exploratory Data Analysis (EDA):
Basic statistical measures like mean, standard deviation, and distribution are computed to understand the dataset.
Correlation Analysis:
A correlation matrix is generated to identify relationships between different features. Heatmaps are used for better visualization.
Regression Analysis:
Linear Regression is applied to model relationships between variables and predict outcomes based on input features.
Classification:
Logistic Regression is used for classification tasks by converting the target variable into binary form and evaluating prediction accuracy.
Visualization:
Various plots such as heatmaps, bar charts, and line graphs are used to represent data patterns and trends clearly.
Statistical Testing:
Z-test is implemented to perform hypothesis testing and analyze statistical significance.
